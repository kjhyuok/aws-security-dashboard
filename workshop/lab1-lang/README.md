# AWS Security Dashboard - 다국어 지원 버전

이 애플리케이션은 AWS 보안 대시보드에 한글/영어 다국어 지원 기능을 추가한 버전입니다.

## 새로운 기능

### 다국어 지원
- 한국어/영어 언어 전환 기능
- 사이드바 상단에 언어 선택 드롭다운
- 모든 UI 텍스트의 실시간 번역
- 세션 동안 언어 설정 유지

### 지원되는 언어
- 한국어 (기본값)
- English

## 실행 방법

1. 가상환경 활성화
```bash
source venv/bin/activate
```

2. 애플리케이션 실행
```bash
streamlit run app.py
```

## 주요 변경사항

### 새로 추가된 파일
- `utils/translations.py`: 번역 데이터
- `utils/i18n.py`: 다국어 지원 유틸리티

### 수정된 파일
- `app.py`: 메인 애플리케이션에 다국어 지원 적용

## 사용법

1. 애플리케이션 실행 후 사이드바 상단의 언어 선택 드롭다운에서 원하는 언어 선택
2. 선택한 언어로 모든 UI가 즉시 변경됨
3. 언어 설정은 브라우저 세션 동안 유지됨

## 기능 테스트

### 언어 전환 테스트
1. 애플리케이션 시작 (기본: 한국어)
2. 언어 선택에서 "English" 선택
3. 모든 UI 텍스트가 영어로 변경되는지 확인
4. 다시 "한국어" 선택하여 한국어로 복원되는지 확인

### 기존 기능 테스트
1. AWS 계정 정보 입력 또는 인스턴스 프로파일 사용
2. 보안 스캔 실행
3. 각 탭(IAM, CloudTrail, 발견사항, 권장조치)의 내용이 선택한 언어로 표시되는지 확인

## 기술적 구현

### 번역 시스템
- 키-값 기반 번역 시스템
- `get_text(key)` 함수로 번역된 텍스트 반환
- 현재 언어 설정에 따라 자동으로 적절한 번역 반환

### 상태 관리
- Streamlit 세션 상태를 사용하여 언어 설정 저장
- 페이지 새로고침 시에도 언어 설정 유지

### 확장성
- 새로운 언어 추가 시 `translations.py`에 번역 데이터만 추가하면 됨
- 모든 UI 텍스트가 중앙화되어 관리 용이
